## 📖 第一部分：專案企劃核心方案

### 💡 1. 專案核心定位與亮點
面對電動車轉型浪潮，燃油車主常因**充電焦慮、使用習慣差異、車價與維護成本、及對電池壽命的未知感**而卻步。

**TOYOTA EV LifePilot** 是一套部署於 **LINE 官方帳號與官網** 的智慧決策與共情互動系統。它打破了傳統生硬的規格表與靜態試算表，透過台灣在地化的用車情境與持有成本精算，在線上消除車主的所有疑慮；並透過 **EV Pilot Pass (體驗券)**，將精準的顧客畫像傳遞給實體展間業務，實現無縫的銷售閉環。

### 🛠️ 3. 三大技術模組設計
*   **📊 模組一：智慧 TCO 與能耗預測器 (TCO Predictor)**：基於和泰去識別化 CDP 數據訓練之 **XGBoost 預測模型**。動態精算燃油車 vs bZ4X 的 5 年持有花費（含折舊、保養、稅金與油電差價）。將省下的金額具體比喻為「幫全家賺回一趟歐洲雙人豪華遊基金」，直擊精算型車主痛點。
*   **🗺️ 模組二：虛擬一日電車生活模擬器 (EV Route Simulator)**：結合台灣地形坡度與氣溫的物理能耗模型。動態模擬「竹科日常通勤（平路）」、「北宜公路礁溪泡湯（利用排檔座旁 **Regeneration Boost 按鍵** 進行下坡動能回充）」及「國道長途台南出遊（中途於清水服務區進行 **CCS1 快充 15 分鐘** 補電節奏）」。
*   **💬 模組三：共情角色智慧顧問 (Empathetic GenAI Consultant)**：Llama-3-Taiwan + 向量資料庫 (RAG) 檢索。透過語意分析識別車主角色（滿分爸爸、科技先鋒、精算贏家），以極具溫度的客製化口吻與擬人化比喻，誠實解答電池保固、保險、稅金與折舊等防禦性疑慮。

### 🔄 4. 無縫 O2O 顧客服務旅程 (User Journey)
1.  **線上體驗**：顧客於 LINE 官方帳號進行 TCO 精算、路況模擬與 AI 共情諮詢。
2.  **一鍵轉換**：滿意後生成專屬 **EV Pilot Pass 體驗券**，內含真實可掃描之 QR Code（加密車主畫像與數據）。
3.  **線下接軌**：展間業務掃描體驗券，平板電腦瞬間同步客戶用車焦點（如後座空間、電池安全性），提供精準客製化實車體驗，縮短成交猶豫期。

#### **方法二：Client-Server 聯調運行模式 (真實前後端連線，最專業)**

## 📈 第三部分：明天收到 CDP 數據後的推進指南

當您明天收到主辦單位寄送的「去個資化 CDP 會員資料」後，請按照以下步驟完成真實模型的訓練與對接：

1.  **資料探索 (EDA)**：
    *   分析歷史車主的持有時間、年里程分佈、回廠保養花費（如機油、火星塞等更換頻率），並理清欄位結構。
2.  **模型訓練**：
    *   在工作區建立一個 Python 腳本，使用 `pandas` 讀取資料，並使用 `scikit-learn` 訓練一個 XGBoost 迴歸模型，以車主用車特徵預測其「5 年燃油車保養與持有總成本」。
3.  **後端對接**：
    *   將訓練好的模型（如 `tco_model.pkl`）儲存於 `backend/` 目錄中。
    *   修改 `backend/main.py` 中的 `predict_tco` 函數，將原本的模擬計算公式替換為：
        ```python
        # 載入模型並進行真實預測
        model = joblib.load("backend/tco_model.pkl")
        predicted_gas_cost = model.predict([[...]])
        ```
4.  **系統聯調**：
    *   啟動後端，打開網頁，此時您的系統將成為一個**具備真實大數據預測能力的成熟 AI 決賽作品**！
